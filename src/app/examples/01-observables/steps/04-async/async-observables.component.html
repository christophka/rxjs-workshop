Observables and async

Take a look at the code in {{ this.file }} and try to implement the existing behaviour using an Observable.

An Observable takes control over producing values and when to emit them to an Observer that consumes these values. As that, values can be emitted async.

No complete

It's totally okay, when your Observable never calls complete.

Make sure to unsubscribe from your Observable.
